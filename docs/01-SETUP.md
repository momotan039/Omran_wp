# Theme Setup Guide

Quick setup checklist for Al-Omran Industries WordPress theme.

## Pre-Installation
- WordPress 5.0+, PHP 7.4+, MySQL 5.6+
- FTP/cPanel access

## Installation Steps

1. **Install Theme**
   - Upload `alomran-theme` to `/wp-content/themes/`
   - Activate in Appearance → Themes

2. **Install Required Plugins**
   - Advanced Custom Fields (ACF)
   - Redux Framework

3. **Configure WordPress**
   - Settings → Permalinks → Select "Post name"
   - Settings → Reading → Set static homepage

4. **Create Pages**
   - About (template: "About Page")
   - Contact (template: "Contact Page")
   - FAQ (template: "FAQ Page")

5. **Create Sample Content**
   - Products: 3+ items with ACF fields
   - News: 2-3 items
   - Testimonials: 2-3 items
   - FAQs: 4-5 items

6. **Configure Menus**
   - Appearance → Menus
   - Add pages and assign to "Primary Menu"

7. **Test Functionality**
   - Homepage, products, contact form, mobile menu

## Optional Configuration
- Chat Widget AI integration
- SEO plugin (Yoast/Rank Math)
- Caching plugin
- Security plugin

## Troubleshooting
- Theme not appearing: Check file permissions, PHP version
- ACF fields not showing: Verify plugin activation
- Styles not loading: Check browser console, clear cache
